Using STM

Create a custom implementation of some data structure. Data structure should work according to its definition, follow the Big-O limitations, be completely thread-safe in concurrent environment and use only STM primitives in implementation.

You are free to use any Java STM library, but we recommend Multiverse STM

Example of STM usage, tests, executable CLI and maven dependency for STM here

Choose datastructure

Select one datastracture listed below, and follow their API.

  • HashMap [interface]
  • DoubleLinkedList [interface]
  • MultiSet [interface]
  • PriorityQueue [interface]
  • DoubleEndedQueue [interface]
  • CircularBuffer [interface]
  • Stack [interface]
  • BinarySearchTree [interface]
  • BinaryHeap [interface]
  • PrefixTree [interface]

You have to write a class which must implemenet interface in every case. Highest performance is not needed.
